## Installing/Running the application

This game requires nodejs to be installed on your computer.

### Install Node Modules

```BASH
    npm install
```
```BASH
    yarn
```

### Start Application
```BASH
    npm start
```
```BASH
    yarn start
```

Then the application will start at localhost:3000

## Application Information
This game is constructed utilizing Mobx, RxJS, and React.
